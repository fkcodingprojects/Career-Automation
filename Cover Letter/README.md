# Cover-Letter-Automation

This Python code automates the process of crafting personalized cover letters for job applications. By inputting key details such as the company name, values, vision, reputation, and culture, the code dynamically inserts this information into a cover letter template. The resulting cover letter is thoughtfully formatted into proper paragraphs, creating a compelling narrative that showcases the applicant's educational background, skills, and experiences in data science, thus demonstrating a strong fit for the specific job role.

## Features

- Automates cover letter writing process
- Dynamically inserts company details into the template
- Formats cover letter into proper paragraphs
- Utilizes ReportLab library to create professional PDFs
- Saves cover letters with company-specific filenames

## How to Use

1. Clone the repository
2. Install required dependencies
3. Run the Python script with appropriate inputs
4. Access the generated PDF cover letter in the output folder
